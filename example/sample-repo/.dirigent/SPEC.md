# Add Greeting and Math Utilities

Add two small utility modules to the sample project:

1. A `greet` module with `hello(name)` and `goodbye(name)` functions
2. A `calc` module with `add(a, b)` and `multiply(a, b)` functions

Both modules should include type hints and docstrings.

## Acceptance Criteria

- `hello("world")` returns `"Hello, world!"`
- `goodbye("world")` returns `"Goodbye, world!"`
- `add(2, 3)` returns `5`
- `multiply(4, 5)` returns `20`
- All functions have type hints and docstrings
- Tests exist for all functions
